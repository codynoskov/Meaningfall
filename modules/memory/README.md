# Memory

Memory governs how product truth becomes and stays usable over time.

Read [`spec.md`](spec.md) for the normative contract.

## Role

- ProductMD owns the `PRODUCT.md` format.
- Memory owns product-truth-over-time governance.
- Guide owns guided creation.
- Install owns install mechanics.

## Boundaries

Memory is not an installed `product-memory/` folder and does not install public
repository structure into user projects.
